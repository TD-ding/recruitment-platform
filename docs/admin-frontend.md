# 管理员前端文档

## 概述

管理员后台是独立的 React 应用，运行在 5174 端口，仅限管理员角色访问。

## 页面

| 页面 | 路由 | 说明 |
|------|------|------|
| 登录 | `/login` | 仅管理员邮箱可登录 |
| 仪表盘 | `/` | 统计概览（用户数、职位数、待审核数） |
| 用户管理 | `/users` | 列表/筛选/删除 |
| 职位审核 | `/jobs` | 审核待审职位（通过/拒绝） |
| 企业审核 | `/companies` | 审核待审企业 |
| 系统设置 | `/settings` | 站点配置项 |

## 布局

- 左侧深色侧边栏导航
- 右侧内容区
- 侧边栏底部退出按钮
- 所有页面需要管理员认证

## 安全

- Token 存储在 `admin_token`（与求职者端隔离）
- 登录时验证 role === 'admin'
- 未认证自动跳转 `/login`
